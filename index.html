Travail en entreprise

Je suis en charge de participer au développement de projet de site internet sur-mesure pour des clients. L'entreprise possède un Framework interne basé sur le principe MVC. Je travaille notamment avec du PHP pour le back-end mais également avec du HTML/CSS pour le front-end. De plus, nous utilisons Git afin que chaque développeurs puissent ajouter son travail au projet et que celui-ci soit valider par le lead dev.

Projet web

Pour moi, un projet web est d'abord un besoin qui provient d'un client. Le but des développeurs dans ce projet est de créer et retranscrire les attentes du clients. Le développeur construit le projet et permet de donner vie à celui-ci. Le développeur est en charge de construire une solution optimale afin que celle-ci soit maintenable pour anticiper les éventuelles évolutions souhaitaient par le client.
